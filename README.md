# Neilyst
基于Python和ccxt的全市场量化回测框架

## 1. V0.1简介
V0.1版本于2023.11.10更新完成。

这将是我本人的量化回测框架，主要功能将包括数据获取，数据落盘，指标计算，数据可视化，策略运行，策略评估等。

## 2. 文件简介
* BaseNeilyst：定义了基础的属性
* DataManager：
    * fetcher：用来获取数据
* Analytics：
    * indicators：用来计算指标和可视化指标
* Strategy：用来运行、评估策略
* config：用来保存基本设置
